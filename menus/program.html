Feature Dashboard This page's graph shows the progress of committed work over each quarter that it considers. The planned feature work and the released feature work are both plotted over time, so that the current quarter's progress can be compared to previous quarters.
Warning: This graph takes a long time to compute!
Initiative Dashboard This page's graph shows the progress of work committed towards a single initiative, which can be selected in the app's settings. The planned feature work and released feature work are both plotted over time for the selected quarter.
All Work Dashboard This page's graph shows the progress of all work completed in each of the quarters that it considers. All work (whether it was planned feature work or not) is included; so long as it is completed during the quarter's timebox, it is featured on the graph.
Warning: This graph takes a long time to compute!
Weekly Throughput Dashboard This page's graph shows the number of artifacts that are completed each week for the trailing 52 weeks. Stories and defects can be counted separately, or added together into one data series.
